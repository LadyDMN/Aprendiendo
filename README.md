GITHUB: ELIMINAR UN REPOSITORIO
🔹 Desde la web (paso a paso)
1.	Entra a GitHub y accede a tu cuenta.
2.	Ve al repositorio que quieres borrar.
3.	Haz clic en la pestaña ⚙️ Settings (Configuración).
4.	Desplázate hasta el final de la página.
5.	Busca la sección “Danger Zone”.
6.	Haz clic en “Delete this repository”.
7.	Escribe el nombre completo del repositorio (por ejemplo: usuario/nombre-repo) para confirmar.
8.	Confirma la eliminación.
⚠️ Importante:
•	Esta acción es permanente.
•	Se eliminarán issues, pull requests, wiki, y todo el historial.
•	Necesitas ser el propietario o tener permisos de administrador.
________________________________________
🔹 Desde la app móvil
Actualmente la app de GitHub no permite eliminar repositorios. Debes hacerlo desde el navegador.
________________________________________
🔹 Alternativa por línea de comandos (GitHub CLI)
Si usas GitHub CLI:
gh repo delete usuario/nombre-repo
Te pedirá confirmación antes de eliminarlo.
